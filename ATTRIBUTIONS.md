# Atribuições

Este projeto utiliza bibliotecas open source de terceiros. As licenças e avisos aplicáveis podem ser encontrados nos repositórios e arquivos de licença de cada dependência.

## UI e componentes

- React: https://react.dev/
- Vite: https://vitejs.dev/
- Tailwind CSS: https://tailwindcss.com/
- shadcn/ui (componentes e padrões): https://ui.shadcn.com/
- Radix UI (primitivos): https://www.radix-ui.com/

## Ícones e utilitários

- Lucide (ícones): https://lucide.dev/
- Axios (HTTP): https://axios-http.com/
- class-variance-authority: https://cva.style/
- clsx: https://github.com/lukeed/clsx
- tailwind-merge: https://github.com/dcastil/tailwind-merge

## Mídia e player

- hls.js (HLS): https://github.com/video-dev/hls.js/

## Interação e UI extra

- @dnd-kit (drag and drop): https://github.com/clauderic/dnd-kit
- Recharts (gráficos): https://recharts.org/
- Sonner (toasts): https://sonner.emilkowal.ski/

- Flask: https://flask.palletsprojects.com/
- Flask-JWT-Extended: https://flask-jwt-extended.readthedocs.io/
- Flask-SQLAlchemy: https://flask-sqlalchemy.palletsprojects.com/
